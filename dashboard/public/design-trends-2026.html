03 — Trend

Scroll-Driven Animations

Native CSS animations tied to scroll position — no JavaScript, no IntersectionObserver, no libraries. Part of Interop 2026, now shipping in all major browsers.

🎯

animation-timeline: view()

Elements animate as they enter the viewport. This card faded + slid in as you scrolled.

📊

animation-timeline: scroll()

The progress bar at the top of this page fills based on scroll position. Pure CSS.

🔮

animation-range: entry 10% entry 50%

Controls exactly when the animation starts and ends relative to the viewport crossing.

⚡

filter: blur() in keyframes

Elements can start blurry and sharpen as they reveal — zero performance cost on GPU-accelerated properties.

animation: reveal-in linear both; animation-timeline: view(); animation-range: entry 10% entry 50%; /* start 10% into viewport, complete at 50% */

05 — Trend

Animated Gradient Borders

Using @property to animate conic-gradient angles — creating hypnotic spinning borders, shimmer effects, and focus states that feel alive.

Spinning Conic

@property --angle animates from 0° to 360° with conic-gradient on a 2px padding wrapper.

Fast Rotation

Same technique, 2s cycle. The speed creates different moods — urgent, energetic, alive.

Slow Reverse

8s reverse cycle. Meditative, premium, considered. Speed = personality.

✨ Shimmer Button
@property --angle { syntax: '<angle>'; initial-value: 0deg; inherits: false; } background: conic-gradient(from var(--angle), #818cf8, #c084fc, #fb7185, #22d3ee, #818cf8); animation: rotate-border 4s linear infinite; /* --angle → 360deg */

CSS Anchor Positioning & Popovers

Position elements relative to other elements — natively. No JavaScript positioning libraries, no getBoundingClientRect(). Part of Interop 2026, shipping in Chrome 144+ and Firefox 147+.

::backdrop + blur. Click to open.

Native Popover

This uses the Popover API — no JavaScript event listeners, no z-index wars, no portal hacks. The ::backdrop gets blur(4px) styling.

/* Anchor Positioning — CSS-only tooltips */ .trigger { anchor-name: --trigger; } .tooltip { position-anchor: --trigger; top: anchor(bottom); left: anchor(center); } /* Popover — native HTML, styled with CSS */ [popover]::backdrop { background: rgba(0,0,0,0.6); backdrop-filter: blur(4px); }
