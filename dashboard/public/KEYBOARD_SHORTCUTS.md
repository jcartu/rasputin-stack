# ⌨️ Rasputin Nexus Dashboard — Keyboard Shortcuts

*Power-user navigation inspired by Bloomberg Terminal, VS Code, and Vim*

---

## 🎯 Quick Actions

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Cmd/Ctrl + P` | Quick Page Switcher |
| `Cmd/Ctrl + /` | Global Search (coming soon) |
| `Cmd/Ctrl + B` | Toggle Sidebar |
| `Escape` | Close Modal/Palette |
| `?` | Show This Help |
| `r` | Refresh Page |

---

## 🧭 Navigation (Vim-Style)

Press `g` followed by a second key to navigate:

| Shortcut | Destination |
|----------|-------------|
| `g h` | **H**ome (GPU Monitoring) |
| `g s` | **S**essions (Council) |
| `g m` | **M**odels |
| `g b` | **B**udget & Forecasting |
| `g r` | Session **R**eplay |
| `g k` | **K**nowledge (Second Brain) |
| `g p` | **P**layground (Prompt Testing) |
| `g a` | Sub-**A**gents |
| `g e` | **E**rror Analytics |
| `g l` | **L**atency Stats |
| `g w` | **W**ebhooks |
| `g t` | **T**emplates (Session Recipes) |

## 📜 Scrolling (Vim-Style)

|----------|--------|

| `k` | Scroll Up |

---

## 💡 Tips

- **Mac users:** `Cmd` is used instead of `Ctrl`
- **Sequences:** Type `g` then `h` quickly (within 1 second)
- **First time?** A hint toast appears on first page load
- **Forgot a shortcut?** Press `?` to see this help anytime

---

## 🚀 Why Keyboard Shortcuts?

Power users can navigate 30-40% faster without touching the mouse. This feature is rare in AI agent dashboards — most competitors (Langfuse, Helicone, AgentOps) don't have comprehensive shortcuts.

Bloomberg Terminal, the gold standard for dense information interfaces, relies heavily on keyboard shortcuts. We're bringing that philosophy to AI agent monitoring.
