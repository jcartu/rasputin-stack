# ALFIE Desktop

Minimal Electron wrapper for ALFIE UI.

## Quick Start

```bash
npm install
npm start
```

Requires ALFIE UI running at http://localhost:3000

## Build

```bash
npm run dist:mac    # macOS
npm run dist:win    # Windows
npm run dist:linux  # Linux
```

## Features

- System tray with show/quit menu
- Single instance lock
- Minimize to tray on close
